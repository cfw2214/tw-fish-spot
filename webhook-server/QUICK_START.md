# ⚡ 5 分鐘快速開始

## 1. 本地測試（1 分鐘）
```bash
cd webhook-server
npm install
cp .env.example .env
# 編輯 .env，加入你的 Token
npm start
```

## 2. 推送到 GitHub（1 分鐘）
```bash
cd ..
git add webhook-server/
git commit -m "Add LINE Bot webhook server"
git push origin main
```
